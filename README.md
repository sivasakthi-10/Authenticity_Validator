# Authenticity Validator for Academia

## 📖 Overview
Authenticity Validator is a web-based application that helps verify the originality of academic documents. It analyzes uploaded PDF, DOCX, or TXT files to estimate plagiarism and AI-generated content, then provides an originality score.

## 🚀 Features
- Upload PDF, DOCX, and TXT files
- Plagiarism detection using TF-IDF & Cosine Similarity
- Basic AI-generated content detection
- Originality score calculation
- Responsive web interface

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Machine Learning: Scikit-learn
- PDF Processing: PyPDF2
- DOCX Processing: python-docx
